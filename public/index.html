The Paragraph Tag

The paragraph tag, or the <p> tag, defines a paragraph in your web page. This has a variety of uses including organization and styling options. When you close a paragraph it does a large line break in the page, to separate other paragraphs. Example:

<p>
   This is a paragraph YAY
</p>
<p>
   This is another paragraph -- YAY!
</p>

Would result with:

This is a paragraph YAY

This is another paragraph -- YAY!

The Anchor Tag

A major tag that is used in almost every web page is the anchor or <a> tag. This tag can be used in multiple ways; the most common way being hyperlinks. Hyperlinks are the blue clicky things on websites that bring you to other pages. Anchor tags can also be used to create an "Anchor" on a page, allowing yout to click a link, or press a button, and skip to a specific point or "anchor" on the page. Here's an example of a hyperlink:

<a href="https://google.com/">My Link!</a>

Brings about

My Link

Page anchors are a little bit more complicated. These allow you to skip down to a position on the page by clicking another link. I use it on this page multiple times -- here's an example.

<a href="#example_anchor">My Anchor Link!</a>
...stuff
<a name="example_anchor"></a>
It will skip to here!

Makes

My Anchor Link!
...stuff
...stuff
...stuff
...stuff

It will skip to here!

Images

The image or <img> tag allows you to put images on the page. This image tag comes with a variety of options, allowing you to set the source, width, and height of the image. I prefer to do alignment settings with CSS in the <style> tag. Here's an example of how to use an image:

<img src="/images/banana.jpg" width="600">

Would give you:

As you can see, setting the width to 600 automatically changes the height to match the aspect ratio. The same thing happens if you set the height and not the width. If you want to distort the image, set the width as well as the height.

The Code Tag

I've used the code tag in this web page many times already, now to show you what it does. The <code> tag is used to format a block of text to look like code. Example:

<code>
   This is some code -- YAY!
</code>

Makes it look like:

This is some code -- YAY!

A pretty simple tag, but still pretty useful.
